# Bluetoothctl

## CLI

* Find bluetooth device
```
[bluetooth]# power on
[bluetooth]# agent on
[bluetooth]# default-agent
[bluetooth]# scan on
```
* Pair the device
```
[bluetooth]# pair 00:1D:43:6D:03:26
```
* Connect to the device
```
[bluetooth]# connect 00:1D:43:6D:03:26
```
